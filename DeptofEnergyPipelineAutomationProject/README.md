**Energy Data Pipeline Automation**

**Automating data workflows to deliver faster insights for energy analytics!!**

**Project Overview**

In this project, I built a data pipeline to automate the ingestion, cleaning, and transformation of energy data.

Previously, analysts had to manually process data every quarter — a time-consuming task. This pipeline replaces that process by automating monthly data updates, enabling quicker and more reliable insights.

**Data Sources**

electricity_sales.csv → Sales data across states and sectors\
electricity_capability_nested.json → Nested data on energy capability

**Tech Stack**

Python\
Pandas (data parsing & transformation)\
JSON & CSV processing

**Pipeline Workflow**

Extract data from raw CSV & JSON files\
Clean and standardize inconsistent formats\
Transform nested JSON into structured tables\
Merge datasets for unified analysis\
Prepare data for downstream analytics

**Key Impact**

Reduced manual processing time from days to minutes\
Enabled automated monthly data refresh\
Improved data consistency and reliability\
Delivered faster insights for business teams
